Cybernetic

  • Result: A

The objective of this school assignment was to create a Web application/API vulnerable to the OWASP TOP 10/OWASP API top 10 and to secure it after vulnerabilities were found. Our team of 4 people decided to make an e-commerence API. We developed it using Python Flask-restx and SQL as the database. We also use static code testing tool, Bandit as well as Dynamic security testing tool, OWASP ZAP to test the security of our API. We also hosted in using Amaazon Web Services(AWS).

vulnerabilities and fixes done by me include:

  • Broken Object level Authorisation
  • JSON Web Token Authorisation Check
  • Broken User Authentication
  • JWT Token Authentication
  • Email Verification
  • Multi factor authentication
